# Explore

## Start

```bash
# command help
deno run --allow-read --allow-write cmd.ts [--update[=tree]]
```

## Feature

* `backend`: deno + oak + webSocket
* `frontend`: simple chat
* `app`: flutter
* `web`: vite + react + dva
* `dart_learn`: dart learning
